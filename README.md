# Anitha's Portfolio Website

A personal portfolio website built with **HTML** and **CSS**, showcasing my skills, projects, and contact information as a Full Stack Developer.

---

## 📁 Project Structure

```
portfolio-website/
├── index.html        # Main portfolio homepage
├── contact.html      # Contact form page
└── style.css         # Stylesheet for the entire website
```

---

## 📄 Pages

- **index.html** — Homepage with About Me, Interests, Projects, and Social Media links
- **contact.html** — Contact form with Name, Email, and Message fields

---

## 🎨 Features

- Custom CSS styling using the **Young Serif** Google Font
- Responsive navbar-style header with an **orangered** accent color scheme
- Styled contact form with bottom-border inputs and hover effects
- Color-coded sections (blue for Interests, orange for Projects)
- Consistent footer with social media links

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts (Young Serif)
